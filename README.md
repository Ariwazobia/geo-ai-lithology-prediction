---

## 📖 Project Overview
Lithology identification is essential in petroleum exploration and subsurface characterization.  
Using synthetic well-log measurements, this project builds a **machine learning workflow** capable of predicting rock types:

- Sandstone  
- Shale  
- Limestone  

A **Random Forest Classifier** is trained to learn relationships between geophysical properties and lithological classes.

## ⚙️ Methodology

### 1. Data Generation
Synthetic well-log data were generated to simulate:
- Gamma Ray
- Resistivity
- Porosity
- Density

### 2. Exploratory Data Analysis
Statistical visualization and correlation analysis were performed to understand relationships between geological variables.

### 3. Model Development
A **Random Forest Classifier** was trained using supervised learning techniques.

### 4. Model Evaluation
Performance was assessed using:
- Accuracy Score
- Classification Report
- Confusion Matrix

### 5. Model Interpretation
Feature importance analysis was used to identify the most influential geological predictors.

The model successfully captured relationships between petrophysical measurements and lithology classes, demonstrating the applicability of machine learning methods in geological modeling workflows.
